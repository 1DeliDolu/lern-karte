## âš ï¸ (5) Exception Handling (Fehlerbehandlung)

Beim Programmieren kÃ¶nnen **Ausnahmefehler (Exceptions)** auftreten, z. B. durch falsche Eingaben.

### Syntax:

```python

    # optional: wird ausgefÃ¼hrt, wenn kein Fehler auftritt

    # optional: wird immer ausgefÃ¼hrt (z. B. AufrÃ¤umarbeiten)
```

---

### Beispiel mit Fehlerbehandlung

```python

    print("Berechnung der Durchschnittstemperatur\n")
```
