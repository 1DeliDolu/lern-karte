# 🏭 1.3.6 Betriebliche Produktionsfaktoren, Güterarten und Organisationsmittel unterscheiden [Seite: 89]

Der Abschnitt erläutert die **betriebswirtschaftlichen Produktionsfaktoren** nach **Gutenberg** (Elementarfaktoren und dispositiver Faktor, ergänzt um **Information/Wissen/Rechte**), ordnet **Güterarten** nach mehreren Kriterien, unterscheidet **Güter zur Leistungserstellung** aus Einkaufssicht und stellt **Organisationsmittel** in klassischem und digitalem Kontext gegenüber. Außerdem werden die **ökonomischen Prinzipien** (Maximum-, Minimum-, Optimumprinzip) zur Faktorkombination eingeführt. 

| Begriff                                    | Definition                                                                                                                                                                |
| ------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Produktionsfaktoren (Inputfaktoren)**    | Alle materiellen und immateriellen Mittel/Leistungen, die an der Bereitstellung von Gütern mitwirken.                                                                     |
| **Elementarfaktoren**                      | **Werkstoffe**, **Betriebsmittel**, **ausführende Arbeit** als direkt produktiv wirkende Faktoren.                                                                        |
| **Werkstoffe**                             | Güter zur Produkterstellung (z. B. Roh-, Hilfs-, Betriebsstoffe, Fertigteile).                                                                                            |
| **Betriebsmittel**                         | Technische Ausstattungen wie Gebäude, Maschinen, Anlagen, Werkzeuge.                                                                                                      |
| **Ausführende Arbeit**                     | Menschliche Arbeitsleistung für ausübende Tätigkeiten (nicht leitend/planend/organisierend/kontrollierend).                                                               |
| **Dispositiver Faktor (Geschäftsleitung)** | **Leitung**, **Planung**, **Organisation**, **Kontrolle** zur Kombination der Elementarfaktoren.                                                                          |
| **Zusätzlicher Faktor: Information**       | **Information/Wissen/Rechte** als eigenständiger Produktionsfaktor (insb. Informationswirtschaft).                                                                        |
| **Ökonomische Prinzipien**                 | **Maximum** (mit gegebenen Mitteln max. Ergebnis), **Minimum** (bestimmtes Ergebnis mit min. Mitteln), **Optimum/Extremum** (optimale Kombination bei variablen Größen).  |

---

## 🧩 (1) Betriebliche Produktionsfaktoren

* **Betriebswirtschaftliche Sicht**: Nach **Gutenberg** werden **Werkstoffe**, **Betriebsmittel** und **ausführende Arbeit** als **Elementarfaktoren** verstanden; der **dispositive Faktor** (GL) verbindet sie zur produktiven Einheit und strebt optimale Kombinationen (Art, Güte, Menge, Zeitpunkt, Ort) an. 
* **Erweiterung**: In der **Informationswirtschaft** ist **Information (Wissen/Rechte)** als zusätzlicher Produktionsfaktor zu berücksichtigen. 

### 📐 Ökonomische Prinzipien (für die Faktorkombination)

| Prinzip                      | Ziel                                                        | Beispiel                                                          |
| ---------------------------- | ----------------------------------------------------------- | ----------------------------------------------------------------- |
| **Maximumprinzip**           | Mit gegebenem **Einsatz** maximalen **Erfolg** erzielen.    | Mit 50 € Taschengeld möglichst lange auskommen.                   |
| **Minimumprinzip**           | Festgelegten **Erfolg** mit minimalem **Einsatz** erzielen. | Werbemitteilung mit minimalen Mitteln kommunizieren.              |
| **Optimum-/Extremumprinzip** | **Optimale Kombination** bei variablen Größen finden.       | Angebot mit günstigem **Output/Input** (z. B. TB/Kosten) wählen.  |

## 📦 (2) Güterarten

**Güter** (Mittel zur Bedürfnisbefriedigung) werden nach verschiedenen **Kriterien** unterschieden: 

| Kriterium                | Güterarten                                                                          | Beispiele                                                                                                      |
| ------------------------ | ----------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| **Beschaffenheit**       | **Materielle Güter** / **Immaterielle Güter**                                       | Gebäude, Auto, Computer, Schokolade / Dienstleistungen, Informationen, Rechte                                  |
| **Ort & Zweck**          | **Konsumgüter**: **Verbrauchs-** / **Gebrauchsgüter**                               | Druckerpapier, Strom, Telefondienste / Maschinen, Regale, Werkzeuge, Autos, Bücher                             |
|                          | **Investitionsgüter**: **Betriebsmittel** / **Werkstoffe** / **Erzeugnisse, Waren** | Gebäude, Maschinen, Werkzeuge, Testgeräte / Roh-, Hilfs-, Betriebsstoffe / hergestellte Produkte, Handelswaren |
| **Beziehung zueinander** | **Komplementär** / **Substitutiv** / **Unverbunden**                                | Drucker & Toner / Tintenstrahl- vs. Laserdrucker, Auto vs. Bahn / Lebensmittel & PC                            |
| **Knappheit**            | **Freie** / **Wirtschaftliche Güter**                                               | Atemluft, Naturerlebnisse, Sonnenschein / Sauerstoff (medizin.), Musikdienst, Sonnenbank, Buch                 |
| **Ausschließbarkeit**    | **Private** / **Öffentliche Güter***                                                | Kinovorstellung, Computer, Pauschalreise / Stadtfest, Parkanlage, Leuchtturm, Ratssitzung                      |

* Öffentliche Güter stehen allen zur Verfügung, sind gemeinsam nutzbar und niemand wird ausgeschlossen. 

### 🛒 Güter zur Leistungserstellung (Sicht des Einkaufs)

| Güterkategorie      | Erläuterung / Beispiele                                                                 |
| ------------------- | --------------------------------------------------------------------------------------- |
| **Rohstoffe**       | Hauptbestandteile eines Erzeugnisses (z. B. Eisen für Gehäuse).                         |
| **Hilfsstoffe**     | Nebenbestandteile (z. B. Farbe, Klebstoff, Heftklammern).                               |
| **Betriebsstoffe**  | Nicht direkt einem Erzeugnis zurechenbar (z. B. Schmieröle, Brennstoffe).               |
| **Einbaubauteile**  | Komponenten für die Herstellung (wenn nicht als Roh-/Hilfsstoffe erfasst).              |
| **Handelswaren**    | Güter zum Weiterverkauf an Endverbraucher.                                              |
| **Fremdleistungen** | Externe Dienste, die zur Leistungserstellung notwendig sind.                            |
| **Lizenzen/Rechte** | Marken-, Software-Nutzungsrechte, Patente, Urheberrechte, die zu berücksichtigen sind.  |

---

## 🗂️ (3) Organisationsmittel

**Organisationsmittel** umfassen in **Aufbau-** und **Ablauforganisation** sowohl **Arbeitsmittel** als auch **formale Gestaltungsmittel**; im Zuge der **Digitalisierung** verändern sie sich grundlegend. 

| Kategorie                                       | Beschreibung / Beispiele                                                                                                                                                          |
| ----------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Klassisches Büro – Arbeitsmittel**            | Computer, Büromaschinen, Maschinen, technische Anlagen, Werkzeuge; Büromaterial, Ordner, Hefter, Register, Mappen.                                                                |
| **Klassische Gestaltungs-/Organisationsmittel** | Arbeitsanweisungen, Formular-/Vordruckwesen, Kontenplan/-rahmen, Organigramm, Stellenbeschreibung/-plan, Tagesordnungen.                                                          |
| **Papierloses Büro – digitale Arbeitsmittel**   | Computer mit vernetzten Peripheriegeräten, Kommunikationsschnittstellen, Office-/Organisationssoftware, **DMS/EIM**.                                                              |
| **Digitale Gestaltungsmittel**                  | Alle Gestaltungsmittel digital verfügbar (Organisationspläne, Ordner, Verzeichnisse, Formulare).                                                                                  |
| **Kollaboration**                               | Gestaltungsmittel werden kommuniziert, initiiert, **gemeinsam bearbeitet**, abgestimmt und organisiert; **KVP**, Innovations- und **Projektmanagement** ganzheitlich integriert.  |

---

## ✅ Kernaussagen kompakt

1. **Produktionsfaktoren**: **Elementarfaktoren** (Werkstoffe, Betriebsmittel, ausführende Arbeit) + **dispositiver Faktor** → zielgerichtete Faktorkombination; **Information/Wissen/Rechte** gewinnt als Produktionsfaktor an Bedeutung.
2. **Güterarten**: Systematische Einordnung nach **Beschaffenheit**, **Zweck**, **Beziehungen**, **Knappheit**, **Ausschließbarkeit**; einkaufsbezogen relevante **Güter zur Leistungserstellung** sind klar abgegrenzt.
3. **Organisationsmittel**: Wandel von **klassischen** zu **digitalen** Arbeits- und Gestaltungsmitteln mit Fokus auf **DMS/EIM**, **Kollaboration** und integriertes **Projekt-/Innovationsmanagement**.
